### Project Name: 
- UTiM (Ultimate Task Manager)
### Project Summary:
- A task manager application that allows users to plan their daily activities, manage their tasks, track their progress, and stay motivated.

### Objectives: 
  -	Engaging UI
  -	A reward or progression system that visualizes userâ€™s  achievements
  -	Basic functions of a task manager app such as charts and TO DO lists.

### Scope:
- This project is for task management which naturally includes time management but it is not a calendar app.
  It rewards the users using progression and achievements but it is not a simple game to collect points. It is not a detailed project manager.
### Target Audience: 
- General audience, mostly student looking for effective ways to organize their plan.
### Key Features:
  -	Add task function
  -	Delete task function
  -	Complete task function
  -	Update task function
  -	Reorganizing task schedules function
